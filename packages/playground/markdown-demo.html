Completions

  • Type ::: at the start of a line for container directives (e.g. :::theorem) — they expand to a full :::name … ::: skeleton.
  • Type :: for include (leaf) directives like ::section / ::image.
  • Or type a directive name at the start of a line (e.g. Theorem) for a Python-style indented directive — Name: with an indented body, no closing fence.
  • Inside $…$, completions switch to KaTeX math macros (try \fra → \frac).
  • [text](# completes against ids declared with {#id}.
  • Trigger manually with Ctrl+Space.

Validation

  • Unknown container directives are flagged (see bogusdirective).
  • Colon fences must open and close with the same number of colons; unclosed or mismatched fences are errors.
  • Unrecognized math macros get a subtle hint.
  • Fenced code, HTML comments, and frontmatter are never flagged.

Notes

  • All intelligence comes from @pretextbook/markdown-style-pretext; this page is just the Monaco adapter (see src/monaco-markdown.ts).
